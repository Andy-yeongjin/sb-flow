Claude 데스크톱 앱
완벽 가이드

인터넷 없이도 Claude와 대화하고, 파일을 분석하고, 강력한 MCP 도구를 활용하세요. 데스크톱 앱만의 특별한 기능을 지금 경험해보세요.

주요 기능

왜 데스크톱 앱인가요?

브라우저 버전과 달리, 데스크톱 앱은 더 깊은 시스템 통합과 강력한 도구를 제공합니다.

오프라인 접근

인터넷이 없어도 Claude와 대화할 수 있습니다. 네트워크 환경에 관계없이 언제든 사용 가능합니다.

MCP 지원

Model Context Protocol로 데이터베이스, 파일 시스템, 외부 앱을 Claude와 직접 연동할 수 있습니다.

파일 첨부 분석

PDF, 이미지, 텍스트 파일을 드래그앤드롭으로 첨부하면 Claude가 즉시 내용을 분석합니다.

빠른 단축키

⌘+N(Mac) / Ctrl+N(Win)으로 즉시 새 대화를 시작합니다. 작업 흐름을 끊지 않고 바로 질문할 수 있습니다.

다중 대화 관리

여러 대화를 동시에 열고 주제별로 관리할 수 있습니다. 업무와 개인 대화를 깔끔하게 구분하세요.

트레이 빠른 실행

메뉴바(Mac) 또는 작업표시줄(Win) 트레이 아이콘으로 어디서든 즉시 Claude를 실행할 수 있습니다.

설치 방법

3분이면 설치 완료

운영체제를 선택하면 맞춤 설치 안내를 확인할 수 있습니다.

1

공식 다운로드 페이지 접속 — 브라우저에서 claude.ai/download로 이동합니다.

2

"Windows용 다운로드" 버튼 클릭 — 설치 파일(.exe)이 자동으로 다운로드됩니다.

3

설치 파일 실행 — 다운로드한 .exe 파일을 더블클릭하여 실행합니다.

4

설치 마법사 진행 — 화면 안내에 따라 설치를 완료합니다. (기본 설정 권장)

5

Claude 앱 실행 후 로그인 — Anthropic 계정으로 로그인하거나 새 계정을 만들어 시작합니다.

1

공식 다운로드 페이지 접속 — 브라우저에서 claude.ai/download로 이동합니다.

2

"Mac용 다운로드" 버튼 클릭 — 디스크 이미지 파일(.dmg)이 다운로드됩니다.

3

.dmg 파일 열기 — Finder에서 다운로드한 .dmg 파일을 더블클릭합니다.

4

Applications 폴더로 드래그 — Claude 아이콘을 Applications 폴더로 드래그앤드롭합니다.

5

Claude 앱 실행 후 로그인 — Launchpad 또는 Applications에서 Claude를 열고 Anthropic 계정으로 로그인합니다.

사용 팁

더 스마트하게 활용하기

Claude 데스크톱 앱을 처음 설치했다면 이 팁들을 먼저 확인해보세요.

1

새 대화 단축키

Mac에서는 ⌘+N, Windows에서는 Ctrl+N으로 즉시 새 대화를 시작할 수 있습니다. 작업 중 빠른 질문에 유용합니다.

2

파일 드래그앤드롭

PDF, Word 문서, 이미지 파일을 대화창에 직접 드래그하세요. Claude가 내용을 분석하고 질문에 답변합니다.

3

MCP 서버 연동

설정 → 개발자 → MCP 서버 추가를 통해 데이터베이스, 파일 시스템, GitHub 등 외부 도구를 Claude와 연결할 수 있습니다.

4

트레이 빠른 접근

Mac 메뉴바 또는 Windows 작업표시줄의 트레이 아이콘을 클릭하면 앱 전환 없이 즉시 Claude 창을 열 수 있습니다.
